### Usage:

To install:

```npm init ```

```npm install mysql2 ```

```npm install sequelize ```

```npm install dotenv ```


Commands to run at the root of your project:

```mysql -u root -p ```

```SOURCE db/schema.sql; ```

```exit ```

```npm run seed ```

```npm start```
